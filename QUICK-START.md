# Schnellanleitung für PWM-Lüftersteuerung

## Schritte zur Verwendung
1. **Installation des Moduls:**  Stellen Sie sicher, dass das PWM-Modul installiert ist.
2. **Anschluss der Lüfter:** Verbinden Sie die Lüfter gemäß den Spezifikationen des Moduls.
3. **Konfiguration der Software:** Passen Sie die Software nach Ihren Bedürfnissen an, indem Sie die entsprechenden Parameter festlegen.
4. **Testlauf:** Führen Sie einen Testlauf durch, um sicherzustellen, dass alles ordnungsgemäß funktioniert.

## Weitere Informationen
Für detaillierte Anweisungen und Fehlersuche konsultieren Sie bitte die Dokumentation im Repository.
